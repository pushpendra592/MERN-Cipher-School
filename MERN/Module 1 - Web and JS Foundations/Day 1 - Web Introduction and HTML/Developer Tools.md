## What are Developer Tools

Developer Tools (DevTools) are built-in browser tools that help you inspect and debug websites. It is like looking under the hood of a car.

---

## How to Open DevTools

| Method | Action |
|--------|--------|
| Right-click | Right-click on page then select Inspect or Inspect Element |
| Windows/Linux | Press F12 |
| Mac | Press Cmd + Option + I |

---

## Key Panels

| Panel | Function |
|-------|----------|
| **Elements** | View HTML structure and tag hierarchy |
| **Styles** | Inspect CSS rules applied to elements |
| **Network** | Monitor loaded files, sizes, load times |
| **Console** | View errors, debug JavaScript output |

---

## What You Can Inspect

### HTML Structure

- Complete HTML code of the page
- How elements are organized
- Tag relationships (parent-child)

### CSS Styles

- All CSS rules applied to elements
- Colors, fonts, sizes, spacing
- Which CSS file applies which style

### Loaded Files

- All files downloaded for the page
- Images, scripts, stylesheets
- File sizes and load times

### Console

- Error messages
- JavaScript output
- Useful for debugging

---

## Benefits of Learning DevTools

- Understand how other websites work
- Debug your own code
- Learn CSS and HTML by examining real websites
- Test changes without editing source files
