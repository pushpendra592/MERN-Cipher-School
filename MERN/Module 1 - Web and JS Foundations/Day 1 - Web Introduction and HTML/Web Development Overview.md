## What is Web Development

The process of creating websites and web applications accessible through the internet via a web browser.

### Key Characteristics

| Characteristic | Description |
|----------------|-------------|
| Browser-based | Runs in Chrome, Firefox, Safari, Edge - no installation required |
| Internet-accessed | Users type URL, browser fetches content from server |
| Cross-device | Works on desktops, tablets, smartphones |
| Multi-file structure | Comprises HTML, CSS, JS files plus assets stored on a server |

---

## Frontend vs Backend

### Frontend (Client-Side)

- **What:** The visible portion of a website rendered in the browser
- **Includes:** Text, images, buttons, colors, fonts, layout, animations, interactive elements
- **Technologies:** HTML, CSS, JavaScript

### Backend (Server-Side)

- **What:** Logic and processes running on remote servers
- **Handles:** Database operations, form processing, authentication, complex calculations
- **Technologies:** Node.js, Python, PHP, databases, etc.

### Communication Flow
```
User <--> Frontend (Browser) <--> Backend (Server)
```


---

## Three Building Blocks of Web Development

| Technology | Role | Purpose | Example |
|------------|------|---------|---------|
| **HTML** | Structure | Defines content and layout (skeleton) | Headings, paragraphs, images, buttons |
| **CSS** | Styling | Controls visual design and appearance | Colors, fonts, spacing, responsive layout |
| **JavaScript** | Behavior | Adds interactivity and dynamic features | Click events, form validation, animations |

### Browser Rendering Order

1. Reads HTML (loads structure)
2. Applies CSS (adds styles)
3. Executes JavaScript (enables interactivity)
4. Displays final website
